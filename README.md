## 🎯 О проекте

**Планировщик семейного меню** — это приложение, которое помогает семьям:
- ⏰ **Экономить время** — планируйте меню на неделю за 15 минут
- 💰 **Снижать расходы** — покупайте только то, что нужно (экономия до 20%)
- ♻️ **Сокращать отходы** — минимизируйте выброшенные продукты (до 30%)

### Кому подходит?

- 👨‍👩‍👧‍👦 **Семьям с детьми** — учёт предпочтений каждого
- 💪 **Людям на диете** — гибкий контроль меню
- 💵 **Экономным** — контроль бюджета на питание
- 🌱 **ЗОЖ-энтузиастам** — сбалансированное питание

## ✨ Возможности

### 💡 Ключевые
- Сохранение и переиспользование готовых меню
- Автогенерация списка покупок из меню
- Возможность использования в рецептах других рецептов (вложенность)

### 📦 Управление продуктами
- Каталог продуктов с ценами
- Закупочные единицы (упаковка, пачка)
- Поиск и фильтрация
- Актуализация цен

### 👨‍🍳 База рецептов
- Неограниченное количество рецептов
- Указание количества порций и готового веса
- Поиск по названию и описанию

### 📅 Планирование меню
- Сетка планирования на неделю
- Гибкие типы приёмов пищи (завтрак, обед, ужин)
- Добавление рецептов или отдельных продуктов
- Указание количества порций

### 🛒 Список покупок
- Группировка по категориям
- Расчёт общей стоимости
- Отметка купленных товаров
- Экспорт в текстовый файл .txt  

### 👥 Члены семьи
- Профили с индивидуальными параметрами:
  - Размер порции (%)
  - Диетические предпочтения и ограничения
- Учёт при планировании меню

### ⚙️ Настройки
- Управление членами семьи
- Управление единицами измерения
- Управление категориями рецептов и продуктов

## ❓ FAQ

**Q: Приложение бесплатное?**  
A: Да, абсолютно бесплатное с открытым исходным кодом.

**Q: Поддерживаются ли другие ОС?**  
A: В MVP только Linux. Windows и macOS планируются в будущих версиях.

**Q: Нужен ли интернет для работы?**  
A: Нет, приложение полностью офлайн. Все данные хранятся локально.

**Q: Где хранятся данные?**  
A: В файле `~/.menutor/data/menutor.db`

**Q: Можно ли синхронизировать между устройствами?**  
A: В MVP — нет. Но можно вручную экспортировать/импортировать JSON.
